**Anbernic RGds Official System Modified Upgrade Program v1.0.1**

**1. Features**  
Download and install the modified version upgrade program for the official Linux system online.

**2. Prerequisites**  
(1) Official Linux system version 20260513 or newer  
(2) Wi-Fi internet connection

**3. Installation and Launch**  
(1) Unzip the `https://github.com/cbepx-me/rgds/releases/download/server/Modified_System_Online_Upgradev1.0.1_for_RGds.zip` file to obtain the `Modified_System_Online_Upgrade.sh` script and the `upgrade` folder.  
(2) Copy both the script and the folder to the `ROMS/APPS` folder or the `Ports` folder on the TF card.  
(3) Insert the TF card, power on the device, go to `Applications` → `Apps` or `Ports`, and run `Modified_System_Online_Upgrade`.

**4. Update Process**  
(1) After launching the program, it will automatically check for the latest online update package.  
(2) Press the **A** button on the main interface to confirm the update and start downloading the update file (supports resumable downloads). The interface will display real-time speed, progress percentage, and remaining size.  
(3) After the download is complete, it will automatically decompress and begin installing the upgrade program.  
(4) Once all upgrade operations are finished, the device will automatically restart.  
(5) Note: After restart, the device may boot into Android. Please press the power button, select shutdown, and then power on again to enter the Linux system.
